# Local AI Agent

A local AI assistant built using:

- LangGraph
- Groq
- Gmail API
- Google Calendar API
- DuckDuckGo Search
- RSS News Feed

## Features

- Read & filter emails
- Send emails (human approval required)
- List & add calendar events
- Web search
- Latest news headlines (RSS-based)

## Setup

1. Install dependencies
2. Configure .env
3. Run:

    python main.py
